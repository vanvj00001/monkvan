    前言#

    北宋年间的蓬莱阁庙会，挤满了赶海的百姓。一个穿破道袍的老道士坐在石阶上，围坐着七八个光着脚的小孩，有的揪他的白胡子，有的摸他腰间挂的破葫芦，叽叽喳喳问个不停。他手里转着个豁口葫芦，笑得皱纹都挤到了一块儿：“今天给你们讲讲八仙的故事——这八位神仙啊，有瘸着腿拄铁拐的乞丐，有倒骑毛驴的老头，有穿破蓝衫一脚鞋一脚赤的浪子，有拿荷花的农家女，还有唯一的女仙，他们凑在一起，闹出了八仙过海的大动静，连龙王都得给他们赔礼。”

    本文将用故事性的笔触，带你走进八仙的世界：从他们各自离奇的成仙经历，到八位神仙凑成一伙的缘由，再看他们一起过海时引发的龙宫风波，理解中国民间对“和而不同”的朴素向往。

    八仙专题总述#

    关键词：八仙、中国神话、八仙过海、八仙专题

    一、八仙的由来：从散仙到群体#

    八仙的说法最早出现在唐宋的笔记里，起初只是零散的神仙传说，没有固定的名单。唐朝《太平广记》里记载的“八仙”还有刘海，并没有何仙姑和曹国舅；到了北宋，坊间话本里才慢慢把蓝采和、张果老固定进来；直到明朝吴元泰的《东游记》，才正式定型为铁拐李、汉钟离、张果老、蓝采和、何仙姑、吕洞宾、韩湘子、曹国舅八位，把刘海换成了曹国舅，才有了我们现在熟知的八仙名单。

    他们出身各异：铁拐李是终南山落第书生，汉钟离是汉朝败军之将，张果老是江湖游方的术士，蓝采和是街头卖唱的浪子，何仙姑是广州增城采荷的农家女，吕洞宾是屡试不第的书生，韩湘子是韩愈府上的纨绔侄子，曹国舅是宋仁宗的皇亲国舅。老道士摸着小孩的头说：“你看，不管你是什么出身，是乞丐还是国舅，只要心向善，都能成仙——这就是老百姓最爱八仙的原因，他们不是高高在上的天庭正神，是和我们一样的普通人变的。”

    有次个小娃子问：“那刘海为啥被换下去了呀？”老道士哈哈大笑：“刘海那是后来的财神，和这八位凑不到一块儿去，曹国舅本来就是皇亲，换进去刚好凑成八个，刚好能坐满一张八仙桌，你说巧不巧？”

    二、各自的绝活：各显神通的底气#

    八仙原本都是散仙，各修各的道，谁也不挨谁。直到有一天，吕洞宾在岳阳楼喝酒，碰到汉钟离也在那儿喝，两人聊了半天，发现脾性特别合，吕洞宾说：“我这人最爱喝酒，遇到你这老道，正好搭伴云游。”汉钟离笑着说：“我正愁没人一起云游，你这书生倒是合我脾气。”

    四、一起的故事：八仙过海的风波#

    八仙凑齐后，在蓬莱阁聚会喝酒，桌上摆着从各地搜罗的好酒，铁拐李的葫芦酒，汉钟离的芭蕉扇扇着风，张果老的驴驮着酒坛，蓝采和唱着歌，何仙姑摆着荷花，吕洞宾提着剑，韩湘子吹着箫，曹国舅敲着玉板，热闹得不行。

    吕洞宾喝得满脸通红，拍着桌子说：“咱们都是神仙了，下次过海，别用船，各显神通，怎么样？”七个人齐声叫好。张果老倒骑着驴说：“我倒骑驴走，回头看海景。”蓝采和挎着花篮说：“我的花篮能飘在海面上，还能装海鲜。”何仙姑拿着荷花说：“我的荷花能浮水，还能遮太阳。”铁拐李拄着铁拐说：“我的铁拐当船，稳得很。”汉钟离摇着芭蕉扇说：“我的扇子能扇风，推着我走。”吕洞宾提着纯阳剑说：“我的剑能斩浪，开路！”韩湘子拿着箫说：“我的箫声能引路，跟着我走。”曹国舅敲着玉板说：“我的玉板能平浪，保平安。”

    一、我的身世#

    1.1 玉帝的外甥#

    我叫杨戬，这个名字陪伴我走过了无数风雨岁月。很多人都知道我是玉皇大帝的外甥，却很少有人知道，这个身份带给我的从来不是荣耀，而是沉重的枷锁和命运的折磨。

    我的母亲是云华仙女，玉帝的亲妹妹。用凡间的话说，她是天庭的公主，身份尊贵无比。可她偏偏爱上了一个凡人——我的父亲杨天佑。父亲不是什么王侯将相，只是一个普通的读书人，在灌江口过着清贫的日子。他善良、正直，对母亲温柔体贴，两人在人间成了亲，生下了我和我的哥哥。

    我至今记得母亲抱着我的样子，她的笑容比天上的云霞还要温暖。那时候我还不知道什么天条禁令，不知道母亲为了这个家承受了多大的风险。我只知道，每天傍晚父亲从外面回来，母亲就会在院子里摆上茶，一家人坐在桃树下说说话——那是我一生中最幸福的时光。

    1.2 天条拆散了我的家#

    可是好景不长。玉帝得知妹妹与凡人私通的消息后勃然大怒。天条写得清清楚楚——神仙不得与凡人通婚，违者天诛地灭。母亲是玉帝的亲妹妹，没有人敢对她动刑，但玉帝照样不留情面。他派天兵天将下凡，强行带走了母亲，把她镇压在桃山之下。

    我记得那一天。天兵天将从天而降，金光刺眼，母亲回过头来看我最后一眼，眼中满是不舍与绝望。她想说什么，却被天兵封住了嘴。我拼命地跑过去想要拉住她的手，却被一股无形的力量震飞出去。等我从地上爬起来的时候，母亲已经不见了，只留下地上一道长长的拖痕。

    父亲抱着我哭了一整夜。第二天，他开始到处奔走求助，想方设法要救出妻子，但没有一个人敢与天庭作对。不到半年，父亲就郁郁而终了。我和哥哥成了孤儿，在灌江口流浪讨饭，风餐露宿。

    从那时起我就暗暗下定决心——不管天庭有多强大，不管天条有多严厉，我都要救出母亲。这是我活着的唯一理由。

    1.3 拜师玉鼎真人#

    在我最落魄的时候，命运终于给了我一线曙光。一位名叫玉鼎真人的老道在灌江口遇到了我，他上下打量了我一番，说你骨骼清奇、根骨不凡，可愿随我上山修行？

    我跪下来就磕了三个响头。不是因为我知道他有多厉害，而是因为我已经无路可走。一个孤儿，还有什么可犹豫的呢？

    玉鼎真人把我带到了玉泉山金霞洞。刚开始的日子并不好过——凌晨就要起来打坐练气，白天要挑水砍柴研习法术，晚上还要背诵密密麻麻的道经。我没有一刻松懈，因为我心里始终有一团火在烧。每次练功累到趴在地上爬不起来的时候，我就想想母亲被压在山下的痛苦，咬咬牙又站起来继续练。

    真人看我的眼神越来越欣慰。有一天他对我说：杨戬，你的天赋远超我的想象，但我教你的只是基本功。真正的神通，需要你自己去领悟。你的体内流着神仙的血脉，你的潜力比任何人都要深。

    1.4 我的哥哥与灌江口#

    说起来，我还有一个哥哥，叫杨蛟。劈山救母之后，天庭来追捕我们，哥哥为了保护我和母亲，死在了天兵天将的刀下。他的死一直是我心底最深的痛。

    灌江口——那个让我又爱又恨的地方。那里是我出生的地方，是我失去母亲的地方，也是我父亲含恨而死的地方。后来我成了灌江口的显圣二郎真君，在灌江口建了庙，把家安在了那里。有人问我为什么不回天庭住，我笑了笑没说话。天庭金碧辉煌，但那里没有我想要的温暖。灌江口虽然只是个小镇，但那里有我的根，有我父亲的坟，有母亲曾经的笑声。

    二、我劈山救母#

    2.1 桃山下的母亲#

    学艺多年之后，我终于打听清楚了——母亲被镇压在桃山之下。那桃山不是普通的山，是玉帝用天界神力凝聚而成的囚牢，山中布满了禁制与符咒，寻常神仙根本靠近不了。

    我偷偷潜到了桃山脚下。那山并不算高，但每一块石头都散发着刺骨的寒意。我试着靠近，结果刚一触碰到山体就被一股强大的力量弹开，手臂上多了一道深深的血痕。那是天条的封印力量——玉帝用天条做锁，把母亲囚禁在山底。

    我围着桃山走了一天一夜，试图找到封印的薄弱之处。终于在一个隐蔽的山谷里，我感知到了母亲的气息——微弱、断续，但确实是她的。我把耳朵贴在山壁上，隐约听到了一个声音在低声哼唱。那是我小时候母亲哄我睡觉时唱的歌谣。

    我的眼泪止不住地流了下来。

    我在山脚下跪了三天三夜。我不知道自己在跪什么——是天庭吗？是玉帝吗？还是那个冷漠无情的天条？我只知道，我要救我的母亲，哪怕赔上这条命。

    2.2 力劈桃山#

    第四天，我站起身来，擦干了眼泪。我知道，哭是没用的，跪也是没用的。天下没有任何人能帮我，我只能靠自己。

    我向玉鼎真人借来了开山斧。真人看着我，沉默了很久，最终叹了口气：杨戬，你可想好了？这一斧下去，你就不只是杨戬了，你将是天庭的叛徒、天条的挑战者。你确定要这么做吗？

    我点了点头。

    回到桃山，我站在山脚下，举起了开山斧。天兵天将们从四面八方涌来，他们大喊着住手、放肆、不要自取灭亡。我深吸一口气，什么也没说，用尽全身力气挥下了那柄巨斧。

    轰——

    桃山从中间裂开了。山石崩塌，烟尘弥漫，一道金光从裂缝中喷涌而出。我扔掉斧头，跌跌撞撞地朝裂缝中跑去，一边跑一边喊：娘！娘！你在哪儿？

    烟雾中传来一个虚弱的回声：是……是戬儿吗？

    我循着声音拨开碎石，终于看到了母亲。她瘦得只剩下皮包骨，脸色苍白得像纸一样，但她的眼睛还是那么温柔，那么亮。我扑过去抱住她，哭得像个三岁的孩子。

    我说：娘，我来了，我来救你了。

    母亲也哭了。她用瘦弱的手摸着我的脸说：戬儿长大了，我的戬儿长大了。

    2.3 天兵天将的追杀#

    但我还没来得及高兴太久，天兵天将就到了。领兵的是玉帝麾下的四大天王，带了三万天兵，把桃山团团围住。

    我护在母亲身前，三尖两刃刀横在胸前。四大天王劝我说：杨戬，放了云华仙女，随我们回天庭请罪，玉帝念在你是他外甥的份上，或可饶你一命。

    我冷冷地笑了一声：饶我一命？我犯了什么罪？我只是救了我的母亲！你们告诉我，救自己的母亲是罪吗？

    没人回答我。

    那一战打得天昏地暗。我一个人对三万天兵，四大天王轮番上阵。我浑身上下都是伤，左臂断了，右腿被刺穿了，血流了一地。但我就是不倒。母亲在旁边哭着喊我走，但我怎么能走？我走了，她怎么办？

    后来是玉鼎真人及时赶到，用元神传音和玉帝沟通了一番。又因为天庭那边正缺人手对付孙悟空和其他反叛势力，玉帝最终松了口——可以不追究我劈山之罪，但有一个条件：我要为天庭效命，做天庭的战将。

    我跪下来，朝玉帝的方向遥遥磕了一个头。我不是在谢他饶了我，而是谢他终于放过了我母亲。

    母亲被接到了天庭瑶池，虽然不得自由行动，但总算不用再受那山压之苦了。我想，这大概就是我能争取到的最好结果了。

    三、我开天眼#

    3.1 第三只眼的由来#

    很多人问我的天眼是怎么来的。有人说天生就有，有人说是在玉泉山修行时开悟的，还有人说是我劈山救母后玉鼎真人帮我开的。这些说法都不全对。

    我的天眼，是在我最痛苦的时候觉醒的。

    那是劈山救母之后，我虽然在桃山大战中活了下来，但体内的伤势极其严重，经脉碎裂了大半，连站起来都困难。玉鼎真人把我带回玉泉山，用尽各种灵药为我续命。我躺在金霞洞里，浑身缠满了绷带，意识模糊不清。

    有一天夜里，我发起了高烧。我迷迷糊糊做了一个梦——梦里我又回到了桃山，我拼命地劈山，可那座山怎么也劈不完，越劈越大，越劈越高。母亲在山的另一边哭喊，我急疯了，使出了浑身的力气去推那座山。

    就在那一刻，我的眉心——也就是两眉之间的印堂穴——突然像被一把烧红的铁烙刺穿了一样，剧烈的疼痛让我从梦中惊醒。我感觉眉心有什么东西正在往外挣，像是有第三只眼睛要从皮肤下面钻出来一样。

    玉鼎真人冲到床边，看到我的样子先是一愣，然后哈哈大笑起来：好！好！你的天眼觉醒了！

    3.2 看穿一切虚妄#

    天眼刚打开的时候，我看什么东西都是模糊的、扭曲的。光线从各个角度涌进我这第三只眼睛，我感觉整个世界都变得不一样了——我能看到墙壁后面的人在做什么，能看到地底下有什么矿藏，甚至能看到天空中云层以上的神仙在飞来飞去。
